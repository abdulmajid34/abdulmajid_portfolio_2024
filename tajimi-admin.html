Sebuah aplikasi Web App Admin Dashboard yang mengelola semua produk makanan dan minuman yang masuk ke aplikasi serta nantinya akan di deliver ke aplikasi Shop Tajimingo app untuk ditampilkan. Tajimi Admin Web App terdapat fitur - fitur seperti management produk, management order, management customer, pembayaran yang masuk, management toko dan pembayaran yang masuk ke aplikasi dan terdapat fitur yang di tampilkan dengan grafik chart. Aplikasi Tajimi Admin Web App ini di development oleh 4 orang, diantaranya 2 orang Back-end developer, dan 2 orang Front-end developer. Saya berposisi sebagai Front-end developer yang bertanggung jawab untuk mendeliver semua api atau integrasi dari Back-end (SERVER) ke Front-end yang nanti nya saya olah menjadi sebuah data dan menampilkannya, tidak hanya itu saya juga bertanggung jawab untuk mengelola semua layout pada aplikasi tersebut.
