Sebuah aplikasi Entry System App yang mengelola pelanggan yang ingin berlangganan listrik dan gas dikota okinawa, Jepang. Entry System App ini terdapat fitur - fitur seperti form data entry pelanggan, bukti kelayakan pelanggan, list harga, pembayaran dengan menggunakan payment gateaway atau kartu kredit dan sudah realtime. Aplikasi Entry System App ini di development oleh 2 orang, diantaranya 1 orang Back-end developer, dan 1 orang Front-end developer. Saya berposisi sebagai Front-end developer yang bertanggung jawab untuk mendeliver semua api atau integrasi dari Back-end (SERVER) ke Front-end yang nanti nya saya olah menjadi sebuah data dan menampilkannya, tidak hanya itu saya juga bertanggung jawab untuk mengelola semua layout pada aplikasi tersebut dan saya juga membantu pekerjaan back-end untuk membuat api di bagian ADMIN web app.
