Sebuah aplikasi web app admin Dashboard untuk mengelola data - data karyawan, sumber daya manusia (SDM), lowongan pekerjaan dan management keuangan. HRIS Admin Web ini adalah aplikasi instansi lembaga pemerintahan yang dibawah naungan Badan Pendapatan Daerah (BAPENDA). Aplikasi HRIS Admin Web ini di development oleh 3 orang, diantaranya 1 orang Back-end developer, 1 orang Fullstack Developer, dan 1 orang Front-end developer. Saya berposisi sebagai Front-end developer yang bertanggung jawab untuk mendeliver semua api atau integrasi dari Back-end (SERVER) ke Front-end yang nanti nya saya olah menjadi sebuah data dan menampilkannya, tidak hanya itu saya juga bertanggung jawab untuk mengelola layout pada HRIS Admin Web ini.
