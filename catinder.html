Catinder App

Description

  • Catinder Web App

  • https://catinder-app.web.app/

  • 03 Juni, 2021

Sebuah aplikasi Web App yang membantu para pencinta cat lovers dalam menemukan pasangan untuk kucing peliharaannya agar dikawinkan. Catinder App ini terdapat fitur - fitur seperti fitur matching up dengan pemilik kucing yang ada di aplikasi, api geolocation untuk mencari keberadaan pengguna dengan range jarak tertentu, realtime chat untuk pelanggan, list kontak pengguna, profile. Aplikasi Catinder App ini di development oleh 4 orang, diantaranya 2 orang Back-end developer, dan 2 orang Front-end developer. Saya berposisi sebagai Front-end developer yang bertanggung jawab untuk mendeliver semua api atau integrasi dari Back-end (SERVER) ke Front-end yang nanti nya saya olah menjadi sebuah data dan menampilkannya, tidak hanya itu saya juga bertanggung jawab untuk mengelola semua layout pada aplikasi tersebut.
